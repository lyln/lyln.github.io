Centos 升级mysql5.1到5.6

系统：CentOS release 6.5

升级前mysql版本：5.1.73-log

升级后mysql版本：5.6.25

升级原因

MySQL 5.1.x不支持四个字节的utf-8编码，只有5.5.x以后才支持

准备

停止mysql

升级mysql
