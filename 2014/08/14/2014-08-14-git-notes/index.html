账户公钥与项目公钥区别

账户设置的 SSH 公钥是用来认证用户身份的，而项目部署公钥是用来授权项目的只读权限的。
账户 SSH 公钥是跟用户账户关联的公钥，一旦设置，就拥有账户下所有项目仓库的读写权限。
项目部署 SSH 公钥是跟项目关联的公钥，设置后有该项目的仓库的只读权限。

账户公钥相当于设置管理员权限一样，你可以clone、pull、[push]
项目公钥，为了安全起见只允许，clone和pull
